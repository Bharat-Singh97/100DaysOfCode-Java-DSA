# Day 09 - Container With Most Water

## 📝 Problem
Find two lines that can store the maximum amount of water.

## 💡 Approach
- Used Two Pointer technique.
- Moved the pointer with the smaller height because it limits the container capacity.

## ⏱️ Time Complexity
O(n)

## 💾 Space Complexity
O(1)

## 🧠 Pattern
Two Pointer

## 🚀 Key Learning
- The smaller height determines the water level.
- Always move the limiting pointer.
