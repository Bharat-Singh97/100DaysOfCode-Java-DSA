# Day 06 - Maximum Subarray

## 📝 Problem
Find the contiguous subarray with the maximum sum.

## 💡 Approach
- Used Kadane's Algorithm.
- Reset current sum whenever it became negative.

## ⏱️ Time Complexity
O(n)

## 💾 Space Complexity
O(1)

## 🧠 Pattern
Kadane's Algorithm

## 🚀 Key Learning
- Greedy approach can solve maximum subarray efficiently.
