# Day 08 - Longest Substring Without Repeating Characters

## 📝 Problem
Find the length of the longest substring without repeating characters.

## 💡 Approach
- Used Sliding Window.
- Expanded and shrank the window whenever duplicates appeared.

## ⏱️ Time Complexity
O(n)

## 💾 Space Complexity
O(n)

## 🧠 Pattern
Sliding Window

## 🚀 Key Learning
- Sliding Window is useful for contiguous substring problems.
