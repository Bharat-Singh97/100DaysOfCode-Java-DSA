# Day 05 - Best Time to Buy and Sell Stock

## 📝 Problem
Find the maximum profit by buying and selling one stock.

## 💡 Approach
- Tracked the minimum buying price.
- Updated the maximum profit while traversing the array.

## ⏱️ Time Complexity
O(n)

## 💾 Space Complexity
O(1)

## 🧠 Pattern
Array Traversal

## 🚀 Key Learning
- Maintain minimum value while maximizing profit.
