# Day 03 - Climbing Stairs

## 📝 Problem
Find the number of distinct ways to reach the top of the stairs.

## 💡 Approach
- Used Dynamic Programming.
- Each step depends on the previous two steps.

## ⏱️ Time Complexity
O(n)

## 💾 Space Complexity
O(1)

## 🧠 Pattern
Dynamic Programming

## 🚀 Key Learning
- Break large problems into smaller subproblems.
- Fibonacci pattern appears in many DP problems.
