# Day 04 - Minimum Capacity Box

## 📝 Problem
Find the minimum capacity required to satisfy the given condition.

## 💡 Approach
- Applied Binary Search on the answer.
- Checked whether a capacity is feasible.

## ⏱️ Time Complexity
O(n log n)

## 💾 Space Complexity
O(1)

## 🧠 Pattern
Binary Search on Answer

## 🚀 Key Learning
- Binary Search can be applied to answers, not only sorted arrays.
