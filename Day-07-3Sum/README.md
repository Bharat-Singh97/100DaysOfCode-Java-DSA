# Day 07 - 3Sum

## 📝 Problem
Find all unique triplets whose sum equals zero.

## 💡 Approach
- Sorted the array.
- Fixed one element and applied Two Pointer technique.

## ⏱️ Time Complexity
O(n²)

## 💾 Space Complexity
O(1) (excluding output)

## 🧠 Pattern
Sorting + Two Pointer

## 🚀 Key Learning
- Sorting simplifies duplicate handling.
- Two Pointer reduces unnecessary comparisons.
