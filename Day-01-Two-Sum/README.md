## 📝 Problem
Given an array of integers and a target value, return the indices of the two numbers whose sum equals the target.

## 💡 Approach
- Used HashMap to store visited elements.
- For each element, checked whether its complement already exists in the HashMap.
- If found, returned the indices immediately.

## ⏱️ Time Complexity
O(n)

## 💾 Space Complexity
O(n)

## 🧠 Pattern
HashMap

## 🚀 Key Learning
- HashMap provides O(1) average lookup.
- Complement technique reduces time complexity from O(n²) to O(n).
